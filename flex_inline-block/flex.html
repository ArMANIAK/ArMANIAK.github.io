Flex mode VS Inline-block mode

Flex mode для html

Как написал один веб-разработчик в ссылке, которую дала Татьяна в телеграмме:"Центрирование чего-либо в CSS - настоящая заноза в заднице.

Есть куча вариантов, как отцентровать/поровнять элементы. Есть куча свойств, которые фиг запомнишь, и которые смешиваются в голове у начинающего верстальщика, стоит лишь на секунду зазеваться. Поэтому в качестве тренировки и домашнего задания я решил сделать подсказку по двум вариантам: Flex иinline-block. Эта страница посвящена варианту Flex.
    В режиме Flex у блоков есть следующие свойства, которые позволяют центрировать/ровнять содержимое:
  1. justify-content предназначен для выравнивания блоков по основной оси
  2. align-items выравнивает блоки по поперечной оси
  3. align-self выравнивает один элемент по поперечной оси
  4. align-content предназначен для контроля пространства между строчками в поперечной оси
