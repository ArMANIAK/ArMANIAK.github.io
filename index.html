These exercises tend to be sort of a practice in JavaScript. Some Small Games and fun-pieces.
Moving square A square which moves from the mouse pointer in random direction. Nice exercise to practice logic and JS.

And it was easy to answer 'Yes', but if You tried to choose 'No' the button starts running away from pointer. I tried to make something similar to that joke.
Guess the number game Quite classic 'game' of guessing the number. You make Your try and programm say whether You guessed or not, and if not whether Your number is bigger or lesser.
Tic-tac-toe game First try to make something more difficult than one-line programms with some sort of logic. A game for two players (but via one desktop)
Bulls and Cows game I tried to make this game more complicated: both design and logic. I've added sort of UI and controls. I'm not sure whether the logic is the simpliest but it works.
Snake Managed to complete it within 2,5 hours. Although it can become fancier in the sense of design and there are still some bugs I'm glad with the job done
