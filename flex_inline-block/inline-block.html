Flex mode VS Inline-block mode

Inline-block mode для html

Как написал один веб-разработчик вссылке, которую дала Татьяна в телеграмме:"Центрирование чего-либо в CSS - настоящая заноза в заднице.

Есть куча вариантов, как отцентровать/поровнять элементы. Есть куча свойств, которые фиг запомнишь, и которые смешиваются в голове у начинающего верстальщика, стоит лишь на секунду зазеваться. Поэтому в качестве тренировки и домашнего задания я решил сделать подсказку по двум вариантам: Flexи inline-block. Эта страница посвящена варианту inline-block.
    В режиме inline-block у блоков есть следующие свойства, которые позволяют центрировать/ровнять содержимое:
  1. text-align предназначен для выравнивания текста в блочных элементах по горизонтали
  2. vertical-align выравнивает текст в строчных элементах по вертикали
